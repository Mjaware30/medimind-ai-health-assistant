# MediMind AI Health Assistant

MediMind AI Health Assistant is a web application that helps users manage chronic health conditions by combining medication tracking, symptom logging, AI health chat, and personalized weekly insights. It is designed to support better medication adherence and health awareness through a simple, responsive interface.

## About
MediMind was created to support people who manage long-term health conditions and need an easy way to stay on track with daily medicines, symptoms, and health-related questions. The app provides a central place to monitor medication schedules, log wellness data, and receive AI-generated summaries based on the user’s recent health activity.

## Features
- Dashboard with medication and symptom overview.
- Medication tracker with dose scheduling and status updates.
- Symptom logger for pain, energy, mood, and sleep.
- AI health chat for natural-language health guidance.
- AI insights that summarize adherence and symptom trends.
- Dose reminders for upcoming medication times.
- Responsive UI for desktop and mobile use.

## Getting Started
### Prerequisites
- Node.js installed on your machine.
- npm installed with Node.js.
- A browser such as Chrome, Firefox, or Edge.
- Claude API key, if AI features are enabled in your version of the project.

### Installation
- Clone the repository from GitHub.
- Open the project folder in your terminal.
- Install dependencies using `npm install`.

## How to Run
- Start the development server with `npm start`.
- Wait for the terminal to show the local development address.
- Open `http://localhost:3000` in your browser.
- Use the dashboard to explore medication tracking, symptom logging, AI chat, and insights.
- If the app uses environment variables, configure them before starting the server.
